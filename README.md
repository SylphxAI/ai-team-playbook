# AI Team Playbook

A collection of skills for running autonomous AI agent development teams.

## Skills

| Skill | Summary |
|-------|---------|
| [Team Agent](skills/team-agent/) | Coordinator + 4-role agent fleet for autonomous development |
| [Vercel Monorepo](skills/vercel-monorepo/) | Intelligent build skipping for Turborepo + Vercel |
| [Neon Branching](skills/neon-branching/) | Dev/prod database branching |
| [Atlas Migrations](skills/atlas-migrations/) | Declarative schema management |
| [Branch Strategy](skills/branch-strategy/) | Branching model, protection rules, merge strategy |
| [Principle-Based Prompting](skills/principle-based-prompting/) | Writing AI prompts that force genuine thinking |

## Philosophy

These are proven patterns from shipping production apps with AI agent teams. Each skill is self-contained and can be adopted independently.

Skills are not theoretical. Every entry here cost real time or money to learn.
